Vous trouverez dans le main le code de notre programme final, ainsi que le rapport détaillé de ce dernier.
Voici l'organisation de ce dépôt :
    -anciens_codes : dossier où se trouvent les anciens codes ainsi que des implémentations que des membres du groupe 
ont réalisé. 
    -résultats : dossier contenant les graphiques et résultat visuel de notre travail.
    -filtres_resultats : dossier contenant les résultats liés au filtre passe-bas sur différentes images et type d'image.
    -image : dossier contenant les images orginales avant traitement.
